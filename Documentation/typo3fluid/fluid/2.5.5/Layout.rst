.. _typo3fluid-fluid-layout:

======
layout
======


With this tag, you can select a layout to be used for the current template.

Examples
========

::

   <f:layout name="main" />

(no output)

Arguments
=========


.. _layout_name:
name
----

:aspect:`DataType`
   string

:aspect:`Required`
   true
:aspect:`Description`
   Name of layout to use. If none given, "Default" is used.
