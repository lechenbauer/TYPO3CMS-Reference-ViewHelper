===========
be.pageInfo
===========


View helper which return page info icon as known from TYPO3 backend modules
Note: This view helper is experimental!

Examples
========

::

   <f:be.pageInfo />

Output::

   Page info icon with context menu

Arguments
=========


This ViewHelper has no arguments.
