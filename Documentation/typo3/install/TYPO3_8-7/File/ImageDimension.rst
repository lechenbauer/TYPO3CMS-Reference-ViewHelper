===================
file.imageDimension
===================


Get width or height from image file

= Examples =

<code title="Defaults">
<f:file.imageDimension>/var/www/typo3/instance/typo3temp/foo.jpg</f:file.size>
</code>
<output>
170
</output>

Arguments
=========


.. _file.imagedimension_dimension:
dimension
---------

:aspect:`DataType`
   string

:aspect:`Default`
   'width'

:aspect:`Required`
   true
:aspect:`Description`
