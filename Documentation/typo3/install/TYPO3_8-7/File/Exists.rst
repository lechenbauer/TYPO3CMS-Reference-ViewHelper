===========
file.exists
===========


Simple view helper to check if given file is a regular file

Examples
========

Default::

   <f:file.exists file="Absolute-path" />

Output::

   TRUE or FALSE

Arguments
=========


.. _file.exists_file:
file
----

:aspect:`DataType`
   string

:aspect:`Required`
   true
:aspect:`Description`
   Absolute path
